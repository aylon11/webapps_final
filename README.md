# SOCKIT WEBSITE!

## How to run?

1. clone repo

2. cd into repo

3. Install dependencies and libraries
`npm install`

4. Start the server 
`npm start`

5. In a different terminal window, cd into "frontend" folder

6. Install dependencies and libraries
`npm install`

7. Start the client
`npm start`

8. Start your local redis server
`redis-server`

9. In your browser, go to "localhost:3000"
